GitHub Copilot クレジット消費量計算ツール

2026/6/1 に premium request 課金は廃止され、GitHub AI Credits（1クレジット = $0.01）の従量課金制に移行しました。 Chat / CLI / code review の消費量を事前見積もりし、事務局承認プロセスに活用してください。

ℹ️ Chat/CLI では system prompt・tool definitions 等の見えない文脈分として、機能ごとの概算オーバーヘッド（FEATURE_OVERHEAD_TOKENS）を入力トークンに自動加算しています。
ℹ️ ファイルサイズ(KB)からのトークン推定は簡易概算です。Word/Excel/PowerPoint/PDF は圧縮・画像・メタデータ等の影響で実際のトークン量と差が出ることがあります。精度を上げたい場合は本文をテキスト欄へ貼り付けてください。

入力設定

参照する過去会話ターン数。多いほど履歴分のトークンが今回のリクエストに加算されます。

参照コード・設計書・添付ファイルなど、同じ参照情報を同一セッション内でどの程度再利用するかの想定です。 初回依頼や単発利用では0%、同じ資料を継続して使う場合は25〜50%を目安にします。 CLIで同一セッションを継続する場合のみ、追加で10%を加算します。

Copilot CLIで同一セッション内に同じ参照情報を使い続ける場合のみ選択してください。

Copilot code review

モデルは GitHub 側で自動選択されるため選択欄はありません。

Copilot code review の消費量は、変更行数 × 暫定係数で概算しています。 実際の消費量とは差が出る可能性があります。
